# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in this project, please do
**not** open a public GitHub issue.

Instead, report it privately by emailing:

**opensourcekigali@gmail.com**

Please include:
- A description of the vulnerability
- Steps to reproduce it
- The potential impact
- Any suggested fixes if you have them

We will acknowledge your report within **48 hours** and aim to resolve
confirmed vulnerabilities within **14 days**.

We appreciate responsible disclosure and will credit researchers
who report valid vulnerabilities, if they wish to be credited.
